Test Navbar

Had an issue with the collapsed menu icon showing the full menu when clicked. So I copied the code from one of my dynamic pages to this static page to test for, and find, the problem.

Turned out that the navbar code had a slight change for the newer version of the Bootstrap framework I was utilizing. Was happy to find the fix. Without setting up this page and going through each line of code I'm not sure how else I would have ever found it.

Created using the Python programming language and the Bottle web framework. Also utilizing the Bootstrap 5 web design framework and some Font Awesome icon library icons.
